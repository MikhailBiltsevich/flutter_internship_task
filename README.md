# О проекте

Все экраны и виджеты сосредоточены в одном файле.  
В отдельный файл вынесен тольк Storage приложения как сервис.  
При большем количестве времени можно распределить экраны по отдельным файлам, виджет карточки типа подписка также можно вынести. Можно избавиться от логики в разметке (вынос методов нажатия кнопки из разметки).  
Также можно доработать работу с переменными типа подписки, вынести в константы приложения.
